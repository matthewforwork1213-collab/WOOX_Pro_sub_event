## 3. 화면별 상세

### S1. 출금 완료 화면 (기능 1)

**변경 A — 광고 영역 콘텐츠 분기**

| 상태 | 조건 | 노출 |
|---|---|---|
| A-1 | WOOX Pro 미포함 & 절약액 > 0 & 계산 가능 (1 USDT 미만은 1 USDT로 표시) | 가상 피드백 카드 ("WOOX Pro였다면 +X USDT") |
| A-2 | A-1 불가(역효과/계산불가) & 온보딩 이벤트 활성 | 테더맥스 WOOX Pro 온보딩 이벤트 |
| A-3 | A-1·A-2 모두 불가 | 기존 하우스/거래소 이벤트 (base 폴백) |
| B-1 | WOOX Pro 포함 & WOOX Pro 자체 이벤트 활성 | WOOX Pro 이벤트 (최우선) |
| B-2 | WOOX Pro 포함 & 자체 이벤트 없음 | 테더맥스 온보딩 이벤트 |
| B-3 | B-1·B-2 불가 | 테더맥스 일반 하우스 광고 |

- 가상 피드백 표시 방식: **토탈 세이빙 기준으로 확정**(OI-08, 2026-07-03). "OOO USDT 더 아낄 수 있었습니다" 프레이밍 1종만 시안 필요.
- 복수 거래소(UID) 동시 출금: 역효과 미해당·양수 UID만 합산한 **단일 총액 하나** 표시 + "N개 거래소 기준·일반 등급 기준" 소문자. **UID가 많아도(5개+) 나열하지 않고 총액 하나로 유지**(선택: "내역 보기"로 펼침 — 디자인 옵션).
- 역효과 방지: **타 거래소 토탈세이빙율 ≥ WOOX Pro면 미노출**(WOOX Pro가 엄격히 클 때만).
- 최소 임계값 1 USDT (양수·1 미만은 1 USDT 표기, 0/음수는 base 로직).
- CTA: "WOOX Pro 수수료 비교 보기" → WOOX Pro 거래소 상세.
- 백오피스 'S1 WOOX Pro 가상 피드백' OFF 시: 기존 광고 로직 복귀(가상 피드백 미노출). (2026-07-07: D+30·2개 WOOX 이벤트 게이트 폐지)

**변경 B — 트레블룰 배너 #2**: 결과 카드 영역 **바깥 하단**에 삽입.

### S2. 페이백 미리보기 결과 페이지 (기능 2)

**변경 A — WOOX Pro 인라인 비교 카드** (팝업·딤 없음)

| 상태 | 조건 | 노출 |
|---|---|---|
| 노출 | 타 거래소 선택 & **역효과 미해당(WOOX Pro 토탈세이빙 > 타)** & 정확 비교 가능 & WOOX Pro 미연동 | 기존 결과 카드 **아래** WOOX Pro 비교 카드 인라인 삽입 |
| 미노출 | WOOX Pro 선택 / 역효과(타 토탈세이빙 ≥ WOOX Pro) / WOOX Pro 미지원(레버리지·상품) | 비교 카드 미삽입, 기존 결과 화면만 |

- 비교 카드 구성: 현재 거래소 vs WOOX Pro 대비(차액 USDT + %p). USDT는 기존 캐시백 엔진의 **Monthly Estimated Cashback 공식**(balance·leverage·TIME·Maker/Taker 가중 수수료율·페이백율·보정계수 0.7, PRD §2-A-1)으로, %p는 **명목 토탈세이빙율 차이**(보정계수 미반영, 2026-07-03 확정)로 각각 다른 산식을 사용. 예시(단순화) Zoomex 조건에서 현재 대비 WOOX Pro 기준 월 약 +567 USDT 더 절약.
- 표시 방식은 **토탈 세이빙 기준으로 확정**(OI-08). 시안은 1종만 필요.
- 보정계수는 **0.7 고정 상수**(전 거래소 동일, 하드코딩). **선행 조건**: 어드민 "테더맥스 적용" 필드 이중곱 버그(별도 트랙, OI-10) 수정 전에는 계산 정확도 보장 불가.
- CTA: "WOOX Pro로 시작" → WOOX Pro 거래소 상세 (브랜드 컬러).
- 기준 명시: 카드 하단 "일반 유저/기본 등급 기준" 소문자.
- 접기(collapse) **제공**(OI-05 확정) — 시안에 펼침/접힘 2개 상태 필요. 강제 닫기(X) 없음.
- 노출 빈도: 상시(조건 충족 시). 세션 제한 없음.
- 백오피스 'S2 WOOX Pro 비교' OFF 시: 비교 카드 미삽입. (2026-07-07: D+30·2개 WOOX 이벤트 게이트 폐지)

**변경 B — 트레블룰 배너 #1**: 결과 페이지 **하단**(인라인 비교 카드 아래)에 삽입.

| 문구(텍스트) | KO: **"트레블룰 연동"** / EN: **"Travel Rule Integration"** (그 외 언어는 표준 로컬라이징) |

| 순서 | 로고(앞) → 텍스트(뒤), 전 언어 고정 |
| 스타일 | 절제형 뱃지 (과한 강조 X) |
| 클릭 | **클릭 시 WOOX Pro 거래소 상세 페이지로 이동** (CTA URL = 어드민 온보딩 등록 값, OI-06). 기능 3 개정(2026-07-07). ⚠️ 정적 정보 배너 전제였던 OI-09 법무 리스크 수용 재확인 필요 |
| 노출 | **백오피스 트레블룰 배너 on/off로 제어**(#1·#2 개별, #3·#4·#5 일괄). 2026-07-07: D+30·2개 WOOX 이벤트 게이트 폐지. ⚠️ D+30 폐지는 OI-07 충돌·C레벨 재승인 대기 |

## 6. 참조: 확정/미결

**확정**: 3개 기능 정의, 기능 1 역산 계산 모델(커미션율 미사용, 페이백 데이터 소스=유저 UID 테이블 일 1회 KST 20시 배치), **기능 2 계산 공식=기존 캐시백 엔진의 Monthly Estimated Cashback(Maker/Taker는 수수료율 단계 가중, 페이백율 단일값, 보정계수=0.7 고정 상수)**, 기능 2 인라인 전환+**접기 제공**, 기능 3 배너(로고 하드코딩·문구 KO/EN), 최소 임계값 1 USDT, USDT 단독+다국어+%p, CTA(WOOX Pro 거래소 상세), **복수 UID=양수만 합산 단일 총액(다수도 나열 안 함)**, **역효과 방지=토탈 세이빙율 기준(타 ≥ WOOX Pro면 미노출)**, **노출 제어=백오피스 영역별 on/off 5개(부분 제어 허용, 2026-07-07 변경 — 기존 2개 WOOX 이벤트 OR+D+30 일괄 롤백 폐지, ⚠️ OI-07 충돌)**, **시작일=WOOX Pro 거래소 온보딩 시점**, **비교 대상=온보딩·노출 Y 거래소 전체**, **가입 URL=어드민 등록 값**, **지속=⚠️ D+30 폐지→백오피스 on/off 통제(OI-07 충돌·C레벨 재승인 대기)**, **표시 방식=토탈 세이빙 기준 확정(OI-08, 페이백률 기준 폐기)**, **%p 정의=명목 토탈세이빙율(WOOX Pro)−명목 토탈세이빙율(현재 거래소), 보정계수 미반영, 기능1·2 공통(2026-07-03)**.
