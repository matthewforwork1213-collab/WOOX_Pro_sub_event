## 핵심 정책 반영 (요약)

- **표시 기준 = 토탈 세이빙**(OI-08). 프레이밍 "…더 아낄 수 있었어요"(손실 회피, 서비스기획서 §5).
- **%p = 명목 토탈세이빙율 차이**(보정계수 0.7 미반영). 복수 UID는 기준수수료 가중평균.
- **보정계수 0.7**은 기능 2 예측에만 적용(기능 1 실측 역산에는 미적용).
- **내부 수치 비노출**: 할인율·페이백율·보정계수·커미션율·마진율은 화면·응답 어디에도 없음(REQ-023). 결과값(USDT·%p)만.
- **인라인 비강제**: 팝업·딤 없이 결과 카드 아래 삽입, 접기 제공·강제 닫기(X) 없음(REQ-012).
- **트레블룰 배너**: `[Bithumb] ⇄ [WOOX Pro]` 로고(하드코딩 자리표시, 사이 좌우 화살표 ⇄) → "Travel Rule Integration"(i18n) 순서 고정. **클릭 시 WOOX Pro 상세 이동**(기능 3 개정 2026-07-07, REQ-018 변경 — ⚠️ OI-09 법무 재확인 필요). 5위치, 프로모션 종료 시 전부 미노출.
- **시간 표시**: 배치·판정은 서버 KST/UTC, 화면은 로컬 타임(프로토타입은 정적 표기, NFR-007).

> 목 백엔드는 프로토타입 검증용이며, 실제 엔드포인트·필드명은 API스펙(#7) 확정을 따른다. 숫자 절대값은 문서 산식 기반의 예시값이다.
